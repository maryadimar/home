Last updated: Januari 5th, 2019

Pengertian wordpress

WordPress adalah sebuah aplikasi sumber terbuka (open source) yang sangat populer digunakan sebagai mesin blog (blog engine). WordPress dibangun dengan bahasa pemrograman PHP dan basis data (database) MySQL. PHP dan MySQL, keduanya merupakan perangkat lunak sumber terbuka (open source software). Selain sebagai blog, WordPress juga mulai digunakan sebagai sebuah CMS (Content Management System) karena kemampuannya untuk dimodifikasi dan disesuaikan dengan kebutuhan penggunanya. WordPress adalah penerus resmi dari b2/cafelog yang dikembangkan oleh Michel Valdrighi. Nama WordPress diusulkan oleh Christine Selleck, teman Matt Mullenweg. WordPress saat ini menjadi platform content management system (CMS) bagi beberapa situs web ternama seperti CNN, Reuters, The New York Times, TechCrunch, dan lainnya. Installation

Step 1 : Install Apache2 HTTP Server

instal web server terlebih dahulu (saya asumsikan temen-temen belum menginstalnya).

ketikan perintah berikut

$ sudo apt update
$ sudo apt install apache2

setelah terinstal berikut perintah untuk mengendalikan apache2.

ketikan perintah berikut

$ sudo systemctl stop apache2.service >> ( menghentikan proses apache )
$ sudo systemctl start apache2.service >> ( memulai apache )
$ sudo systemctl enable apache2.service >> ( mengaktifkan apache )

Step 2 : Install MariaDB Database Server

Untuk menyimpan database kita gunakan MariaDB. Untuk menginstalnya jalankan perintah di bawah ini :

ketikan perintah berikut

$ sudo apt-get install mariadb-server mariadb-client

berikut perintah untuk mengendalikan mariaDB, sama halnya seperti proses apache pada step 1 :

ketikan perintah berikut

$ sudo systemctl stop mariadb.service
$ sudo systemctl start mariadb.service
$ sudo systemctl enable mariadb.service

Setelah itu, jalankan perintah di bawah ini untuk mengamankan server MariaDB dengan membuat kata sandi root dan melarang akses root jarak jauh.

ketikan perintah berikut

$ sudo mysql_secure_installation

ikuti tahap-tahapnya

  • Enter current password for root (enter for none): Just press the Enter
  • Set root password? [Y/n]: Y
  • New password: Enter password
  • Re-enter new password: Repeat password
  • Remove anonymous users? [Y/n]: Y
  • Disallow root login remotely? [Y/n]: Y
  • Remove test database and access to it? [Y/n]: Y
  • Reload privilege tables now? [Y/n]: Y

restart MariaDB

$ sudo systemctl restart mariadb.service

Step 3 : Install PHP 7.1 and Related Modules

Disini yang saya instal php versi 7.1 kalau temen2 ingin instal versi yang baru juga silahkan saja.

ketikan perintah berikut

$ sudo apt-get install software-properties-common $ sudo add-apt-repository ppa:ondrej/php

Kemudian lakukan update, perintahnya sebagai berikut :

ketikan perintah berikut

$ sudo apt update

kemudian, ketikan perintah berikut untuk instal PHP 7.1

$ sudo apt install php7.1 libapache2-mod-php7.1 php7.1-common php7.1-mbstring php7.1-xmlrpc php7.1-gd php7.1-xml php7.1-mysql php7.1-cli php7.1-mcrypt php7.1-zip php7.1-curl

setelah itu lakukan perubahan pada file php.ini

ketikan perintah berikut :

$ sudo nano /etc/php/7.1/apache2/php.ini

hal yang perlu di atur pada bagian file php.ini sebagai berikut :

gunakan CTRL + W untuk memudahkan pencarian (jika open menggunakan terminal):

file_uploads = On
allow_url_fopen = On
memory_limit = 256M
upload_max_filesize = 100M
max_execution_time = 360
date.timezone = Asia/Jakarta

Step 4 : Membuat Database

Login terlebih dulu ke MariDB Server

ketikan perintah berikut

$ sudo mysql -u root -p

Kemudian buat database

ketikan perintah berikut

$ CREATE DATABASE wpdb;

Set password

ketikan perintah berikut

$ CREATE USER 'wpdbuser'@'localhost' IDENTIFIED BY 'silahkan_isi_passwordnya';

Kemudian beri pengguna akses penuh ke database

ketikan perintah berikut

$ GRANT ALL ON wpdb.* TO 'wpdbuser'@'localhost' IDENTIFIED BY 'silahkan_isi_passwordnya' WITH GRANT OPTION;

Terakhir, simoan perubahan dan simpan

ketikan perintah berikut

$ FLUSH PRIVILEGES;
EXIT;

Step 5 : download Wordpress terbaru

Silahkan kunjungi situs resmi wordpress untuk mengunduh, jika sudah silahkan lakukan perintah berikut.

ketikan perintah berikut

$ cd /tmp && wget https://wordpress.org/latest.tar.gz
$ tar -zxvf latest.tar.gz
$ sudo mv wordpress /var/www/html/wordpress

Kemudian jalankan perintah di bawah ini untuk mengatur izin yang benar agar WordPress berfungsi.

ketikan perintah berikut

$ sudo chown -R www-data:www-data /var/www/html/wordpress/
$ sudo chmod -R 755 /var/www/html/wordpress/

Step 6 : Konfigurasi Apache2

Terakhir, konfigurasikan file konfigurasi situs Apahce2 untuk WordPress. Untuk mengontrol bagaimana pengguna mengakses konten WordPress. Jalankan perintah di bawah ini untuk membuat file konfigurasi baru bernama wordpress.conf

ketikan perintah berikut

$ sudo nano /etc/apache2/sites-available/wordpress.conf

Kemudian salin dan tempel konten di bawah ini ke dalam file dan simpan. Ganti baris yang disorot dengan nama domain dan lokasi root direktori Anda sendiri.

ketikan perintah berikut

<VirtualHost *:80>
ServerAdmin admin@example.com
DocumentRoot /var/www/html/wordpress/
ServerName example.com
ServerAlias www.example.com

Options +FollowSymlinks
AllowOverride All
Require all granted

ErrorLog ${APACHE_LOG_DIR}/error.log
CustomLog ${APACHE_LOG_DIR}/access.log combined
</VirtualHost>

Simpan file and exit.

Step 7 : Aktifkan Modul WordPress dan Tulis Ulang

Setelah mengkonfigurasi VirtualHost di atas, aktifkan dengan menjalankan perintah di bawah ini ... perintah juga menonaktifkan PHP7.0 dan mengaktifkan PHP 7.1 untuk Apache2.

ketikan perintah berikut

$ sudo a2ensite wordpress.conf
$ sudo a2enmod rewrite
$ sudo a2dismod php7.0
$ sudo a2enmod php7.1

Step 8 : Restart Apache2

Untuk memuat semua pengaturan di atas, mulai ulang Apache2 dengan menjalankan perintah di bawah ini.

ketikan perintah berikut

$ sudo systemctl restart apache2.service

Step 9 : konfigurasi wordpress

jalankan perintah di bawah ini untuk membuat file WordPress wp-config.php.

ketikan perintah berikut

$ sudo mv /var/www/html/wordpress/wp-config-sample.php /var/www/html/wordpress/wp-config.php

Kemudian jalankan perintah di bawah ini untuk membuka file konfigurasi WordPress.

ketikan perintah berikut

$ sudo nano /var/www/html/wordpress/wp-config.php

silahkan atur konfigurasi databasenya.

ketikan perintah berikut

// ** MySQL settings - You can get this info from your web host ** //
/** The name of the database for WordPress */
define('DB_NAME', 'wordpress');
/** MySQL database username */
define('DB_USER', 'wordpressuser');
/** MySQL database password */
define('DB_PASSWORD', 'user_password_here');
/** MySQL hostname */
define('DB_HOST', 'localhost');
/** Database Charset to use in creating database tables. */
define('DB_CHARSET', 'utf8');
/** The Database Collate type. Don't change this if in doubt. */
define('DB_COLLATE', '');
